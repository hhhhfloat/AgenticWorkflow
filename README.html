⚙️ Agentic Workflow

基于 DeepSeek API 的本地 AI 编程助手
用自然语言描述需求，Agent 自动生成或修改代码

📌 环境要求

  • Windows系统
  • 需要安装 Java 21 或更高版本
  • 可下载： Oracle JDK
  • 安装后，在命令行输入 java -version 能正常输出版本信息即可

🚀 如何使用？

  1. 双击 start.bat
  2. 输入你的 DeepSeek API Key（首次运行会提示）
  3. 浏览器自动打开前端页面（本地运行，无服务器）
  4. 在文本框中输入需求，点击 “运行”

📝 v3.0 更新说明

  • 多语言支持：Java（单文件/Maven）、Python、Node.js、C++（MSVC/MinGW）
  • 前端配置面板：无需修改代码，直接在界面中调整 API Key、模型、路径等配置
  • API Key 管理：一键清除 API Key，自动删除环境变量并重启服务

📝 v3.2 更新说明

  • 前端编译支持：Agent 编译成功过的项目，会在前端项目文件夹直接生成一个按钮，一键编译运行
  • 停止功能优化：按下停止后，立刻断连，之后可直接再次运行，无需重启
  • 提示词优化：当创建需要输入的脚本时，agent会自主重定向 - 创建测试文件 - 编译运行测试程序 - 清理重定向代码与测试文件 - 重新编译（但有的时候可能不是那么听话）
  • 注：手动编译运行非图形化项目时，不能通过输入流传入数据，因此对应项目会判断为超时退出。（不过一般agent已经自行测试过了）

📝 v3.3 更新说明

  • 修复了字符串注入问题：原来的用户停止检测使用字符串检测，导致读取的文件中包含对应字符串则会立即停止工作

📂 文件目录说明

  • sandbox/ → Agent 生成的所有代码都在这里
  • TestProjects/ → 归档的完整项目
  • HistoryOutput/ → 每次运行的日志文件
  • 以上几个文件夹会在 .jar 文件同级自动生成

⚠️ 注意事项

  • 需要联网（调用 DeepSeek API）
  • API Key 请从 platform.deepseek.com 官网获取（一点小钱够用挺久了）
  • 建议迭代次数设为 20+
  • 刷新页面会导致断连
  • 关闭命令行窗口即可停止服务
